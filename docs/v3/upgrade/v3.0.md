## 配置文件调整

v3.0 版本由于支持了多租户，所以配置文件有所调整。

请参考 [初始化](/docs/v3/quick-start/init.md) 调整您的配置文件。

## 初始化（非必须，但推荐）

此项非必须，但是强烈推荐您更改调整

同样参考 [初始化](/docs/v3/quick-start/init.md) 的 **方式一** 调整您的配置初始化。

## 更改版本号

将 composer.json 的 yansongda/pay 版本号更改为 `~3.0.0`，随后 `composer update` 即可。
